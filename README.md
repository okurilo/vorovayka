# Виджетрон

MV3-расширение Chrome для захвата DOM, связанных API-запросов и компактного экспорта виджетов без отправки данных наружу.

## Что умеет

- Выбирать DOM-элемент на странице и собирать его snapshot.
- Захватывать релевантные `fetch`/`XHR` для armed-домена.
- Показывать в рабочей области DOM preview, выбранные API и итоговый экспорт.
- Подсвечивать связи `DOM ↔ API`, если для capture доступен `cloneSpec`.
- Давать компактный экспорт в режимах `Всё вместе`, `API`, `API types`, `DOM clean`, `DOM raw`.

## Основной сценарий

1. Откройте `chrome://extensions` и загрузите проект как `Load unpacked`.
2. Откройте обычную страницу по `http` или `https`.
3. В popup `Виджетрон` включите захват для текущего домена.
4. После перезагрузки вкладки нажмите `Выбрать элемент`.
5. Подтвердите нужные API в in-page модалке.
6. Откройте рабочую область и проверьте DOM, API и экспорт.

## Ограничения

- Захват сети работает только для явно armed-домена.
- Данные не отправляются наружу: используются только локальные механизмы расширения.
- `latestCapture` остаётся временным сообщением и очищается после чтения или по TTL.
- Если полный capture недоступен и используется compact fallback, это должно быть явно видно в UI.

## Проверка

- После любых изменений кода запускайте `npm test`.
- Для ручной проверки используйте popup и viewer как основные интерфейсы расширения.
